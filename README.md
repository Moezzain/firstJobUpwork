# E-Commerce
E-Commerce backend using Node &amp; mysql

# to run this repo:
    1-install node.
    2-git clone https://github.com/Moezzain/E-Commerce.git
    3-npm i
    4-npm run dev

#Response codes and Correspondoing Messages:

    0       Success
    9       Wrong Credentails
    10      DataBase Error
    11      Invalid id
